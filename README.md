Games Collection

A modern, visually refined web collection of classic AI and logic games. Each game features a clean, minimalistic design, smooth animations, and a professional user interface. All games run entirely in the browser—no installation required.

Included Games
Water Jug Problem: Solve the classic water jug puzzle.
N-Queens Problem: Place 8 queens on a chessboard so that no two attack each other.
8-Puzzle Problem: Slide tiles to solve the 8-puzzle.
Vacuum Cleaner Problem: Clean all dirt from a grid using a virtual vacuum.
Tic Tac Toe: Play the classic two-player game.

Features
Modern, responsive UI with smooth gradients and soft shadows
Timers for each game
High-quality icons and refined typography
No dependencies—just open `index.html` in your browser
